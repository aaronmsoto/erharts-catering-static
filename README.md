## Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/[your-username]/erharts-catering-static.git
   cd erharts-catering-static
   ```

2. Open `index.html` in your browser or use a local server:
   ```bash
   python -m http.server 8000
   # or
   npx serve
   ```
